Game ticks and subtick order

Everything in Minecraft occurs in increments of 1/20th of a second, called ticks. Everything the game does is processed tick-by-tick. The order of events in the tick is as follows:

  • Update world time
  • Scheduled ticks for blocks
  • Scheduled ticks for fluids
  • Raid handling
  • Mob spawning (natural, followed by spawner blocks)
  • Block events (pistons beginning to push, note blocks making sounds)
  • Entity handling
  • Block entities (pistons moving blocks, pistons finishing their push, hoppers ticking)
  • Player inputs
